# Basic data analysis project.
In this project I will analyse so called "GoFast" company with synthetic data.
Data will be given as ".csv" tables. 

Project contains:

- Data preprocessing
- Statistical analysis
- Data merging
- Hypothesis tests / t-tests


Tools used:

- Python(.ipynb file)
- Pandas, NumPy
- Matplotlib, Seaborn
- SciPy
